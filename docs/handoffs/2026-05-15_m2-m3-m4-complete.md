- Visual Studio 2026 (`C:\Program Files\Microsoft Visual Studio\18\Community`), MSVC v143, C++20, `/MD`, `/W4 /permissive- /utf-8`
- **SDK 강제**: `Enter-VsDevShell ... -DevCmdArguments '-arch=x64 -host_arch=x64 -winsdk=10.0.22621.0'`. SDK 10.0.26100.0은 `specstrings_strict.h`가 누락되어 컴파일 실패.
- Build: `cmake --build build --config Release`
- Tests: `& build\core-tests.exe` (현재 244/244 pass)

### COM apartment

- UI thread: STA (`OleInitialize`)
- Worker (`PaneController`의 jthread, `FsWatcher`의 jthread, RingLogger writer, ProcessMemoryService notifier): 모두 MTA, COM 미사용

### Cross-thread message routing

- Worker → UI: `PostMessageW(hwnd, kWmFe*, gen, payload)`
- WM_FE_* 핸들러는 모두 `isStaleGeneration(wParam)`로 시작 (M4 generation L1)
- `kFsCoalesceMs = 100 ms` SetTimer로 fs-change burst 흡수

1. **AskUserQuestion 강제** — 분기/선택 질문 시
